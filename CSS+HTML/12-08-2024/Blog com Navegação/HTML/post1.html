O Que é o Pudim de Gelatina?

O pudim de gelatina é uma sobremesa leve e refrescante, que leva, em resumo, gelatina, leite condensado, e creme de leite.

Essa é uma versão mais simples do pudim tradicional, que usa a gelatina incolor como base para criar uma textura firme e cremosa.

Ao servir gelado, ele é perfeito para dias quentes ou ocasiões em que se deseja uma sobremesa rápida e prática.
